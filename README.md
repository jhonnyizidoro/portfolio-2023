# Portfolio

Personal portfolio website — showcasing my experience as a software engineer, projects I've shipped, and skills I've picked up along the way.

## Stack

- **Next.js 16** — React framework
- **TypeScript** — type safety throughout
- **SASS** — scoped CSS modules
- **Vercel Analytics** — page view tracking
- **i18n** — English (`en-US`) and Portuguese (`pt-BR`)

## Sections

- **Introduction** — hero with animated role subtitles
- **Skills** — tech stack overview
- **Experience** — career timeline (Fox, Globo, Vizabli, and more)
- **Projects** — selected work with images and links

## Running locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Scripts

| Command | What it does |
|---------|--------------|
| `npm run dev` | Dev server with hot reload |
| `npm run build` | Production build |
| `npm run start` | Serve production build |
| `npm run lint` | ESLint check |
